# ipb-ppki-4

Template untuk penulisan karya ilmiah/tugas akhir menggunakan [Typst](https://typst.app)
yang mengacu kepada [PPKI Edisi ke-4](https://simak.ipb.ac.id/Foto/Panduan/Dokumen_38.pdf).

Berikut jenis karya ilmiah/tugas akhir yang didukung:
- [ ] Laporan Magang
- [ ] Laporan Proyek Akhir
- [ ] Skripsi
- [ ] Tesis
- [ ] Disertasi
